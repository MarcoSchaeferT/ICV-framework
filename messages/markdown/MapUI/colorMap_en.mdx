#### Color Map

Choose a color map that fits your selected feature (table column). The value range is automatically determined, and the selected color map is applied accordingly. A **color legend**, showing the mapping of the values to the colors, is always located in the bottom right corner of a map component. By using the slider on the right side of the dropdown menu, you can adjust the opacity of the color map.

> [!TIP] **Example - choosing a color map:** Choose a diverging color map for temperature values (blue to red) or a linear color map for precipitation values (white to blue).
