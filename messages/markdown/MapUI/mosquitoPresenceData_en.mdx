#### Mosquito Presence Data

This interface element allows to control the visualization of mosquito sightings worldwide. The non-representative dataset includes sightings from 2013 to 2025. Sightings are shown as <span style={{ color: '#c43a3aff', fontSize: '1.2em' }}>●</span> dots on the map, which can be filtered by year and by mosquito species using the dropdown menus. (for further information on the species and vector borne diseases see [Vector Information](./domainInfo))
