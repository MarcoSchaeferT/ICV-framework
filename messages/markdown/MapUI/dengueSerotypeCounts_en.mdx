


#### Dengue Serotype Counts

This element steers the visualization of [dengue serotype](https://en.wikipedia.org/wiki/Dengue_fever#Virology) counts per country using a non-representative data set. Each donut chart shows the distribution of serotypes, with the total count displayed in the label above. The label background color indicates the number of counts in comparison to other countries. (Data source: [Nextstrain](https://nextstrain.org/dengue/all/E))

**Interactions:**
Donut charts can be toggled on/off. Donut chart sizes are adjustable, and hovering over a segment reveals more detailed information. Additionally, dropdowns allow filtering by year and serotype. When a serotype is selected, only charts that include the selected serotype are displayed.
