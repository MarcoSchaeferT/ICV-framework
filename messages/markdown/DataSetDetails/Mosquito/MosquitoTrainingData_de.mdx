#### Trainingsdaten Übersicht

Der Datensatz der für die Bestimmung der Habitateignung verwendet wurde enthält Informationen zu verschiedenen Umweltfaktoren, die die Lebensräume von Mücken beeinflussen. Die Daten liegen in einem Gitterformat vor, wobei jede Gitterzelle (n={"267.876"}) etwa 28 km x 28 km misst (am Äquator). 

Allgemeine Eigenschaften jeder Gitterzelle:
- 0,25° x 0,25° Fläche
- Werte: 
  - jährlich: relative & exakte Werte
  - monatlich: exakte Durchschnittswerte

Jede Zelle enthält folgende Umweltfaktoren:
- monatlich:
  - Temperatur
  - Niederschlag
  - Taupunkt
  - Windgeschwindigkeit
- jährlich:
  - Bevölkerungsdichte (interpoliert von 5-Jahres-Intervallen)
  - Sichtungsdaten
  - Landnutzung 
    - Ozean
    - städtisch
    - Ackerland
    - Weide-/Grünland
    - Wald
    - unbewirtschaftetes Gras-/Buschland
    - spärliche/keine Vegetation
    - Wasser

Diese Daten können im [Training Data View](./rawDataView) eingesehen und erkundet werden.
