#### Training Data Overview
 The mosquito training data set provides information on various environmental factors that influence mosquito habitats. The data is represented in a grid format, with each grid cell (n={"267,876"}) approximately 28km x 28km at the equator. 

General Properties of each grid cell:
- 0.25° x 0.25° area
- values: 
   - yearly: relative & exact values
   - monthly: exact average values

Each cell contains the following environmental factors:
- monthly:
   - temperature
   - precipitation
   - dew point
   - wind speed
- yearly:
   - population density (interpolated on 5-year intervals)
   - presence data
   - land usage
      - ocean
      - urban
      - cropland
      - pasture/rangeland
      - forest
      - unmanaged grass/shrubland
      - sparse/no vegetation
      - water

These data can be inspected and explored in the [ Training Data View](./rawDataView).
