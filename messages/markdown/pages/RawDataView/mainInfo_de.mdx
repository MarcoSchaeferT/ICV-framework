Entdecken Sie die Daten, die zum Trainieren der Machine-Learning-Modelle verwendet wurden. Diese Daten helfen dabei, die Eignung von Lebensräumen für Mücken vorherzusagen, und umfassen Umweltfaktoren wie Klima, Landnutzung und Bevölkerungsdichte. Die gitter-basierten Daten werden auf der Weltkarte (**Kartenansicht**) unten angezeigt. Sie können über das Dropdown-Menü **Daten-Merkmal** zwischen verschiedenen Faktoren zur Visualisierung wechseln. Weitere Details finden Sie in den Infokarten, die Sie aufrufen können, indem Sie mit der Maus über das Symbol **ⓘ** fahren, das sich in der oberen rechten Ecke jeder Komponente befindet.
