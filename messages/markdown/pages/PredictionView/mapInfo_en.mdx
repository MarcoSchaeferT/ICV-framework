### Overview Map

This world map focuses on visualizing grid-based data on the habitat suitability for different mosquito species (see [Vector Information](./domainInfo)). The view also provides mosquito presence data represented as dots showing the exact locations. Additionally, dengue serotype counts are visualized as donut charts in a separate layer.
