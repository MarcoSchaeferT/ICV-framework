### Detailkarte  

Diese Weltkarte konzentriert sich auf die Visualisierung von gitterbasierten Daten zur Lebensraumtauglichkeit verschiedener Mückenarten (siehe [Vektor Informationen](./domainInfo)). Sie dient als *Detailkarte*, auf der ausgewählte Bereiche der **Übersichtskarte** in einem höheren Zoomlevel angezeigt werden.

---------------

### Verwendung  
Diese Ansicht bietet nur begrenzte Interaktionsmöglichkeiten (Klicken, Einblendungen wenn mit dem Cursor über Objekte gefahren wird). Die Ansicht wird hauptsächlich über die **Übersichtskarte** gesteuert. Das bedeutet, dass alle Interaktionen und Anpassungen, die in der **Übersichtskarte** vorgenommen werden, hier übernommen werden. Weitere Informationen finden Sie über das <span style={{ display: "inline-flex", verticalAlign: "middle" }}><InfoIcon size={16} /></span> Symbol der **Übersichtskarte**.
